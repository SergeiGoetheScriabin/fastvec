# FastVec


SIMD-accelerated vector operations for PostgreSQL.

> in progess...


## Overview

FastVec is a PostgreSQL extension written in modern C++ that provides efficient vector storage and similarity search primitives.

Features:

-custom vector data type
-close to L2 cache
-SIMD (AVX2) optimizations
-PostgreSQL operators
-going to benchmark performance ofc..
